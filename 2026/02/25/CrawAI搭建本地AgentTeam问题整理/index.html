关键经验总结

  1. 版本锁定：crewai 0.35 是目前最稳定的本地模型版本
  2. 模型选择：7B 只能跑通流程，生产用必须上 API
  3. 状态保存：每个 Agent 完成立即写文件，支持断点续跑
  4. 内存管理：M3 统一内存看”联动内存”，不是进程 RSS
  5. API 策略：DeepSeek 性价比高，Kimi 上下文长，组合使用
