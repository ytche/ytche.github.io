Java IO流详解

IO概述

Java IO是Java中进行输入输出操作的基础包，包括字节流和字符流两大类。

分类

字节流

  • InputStream：输入字节流
  • OutputStream：输出字节流

字符流

  • Reader：输入字符流
  • Writer：输出字符流

总结
